# App Sections

## Mobile Public

| Section | Purpose | Key restrictions |
| --- | --- | --- |
| Public Home | Explain the app and expose public value | Public content only |
| About the Order | Educate interested people | Requires Order approval |
| Prayer Library | Public prayer book | Public prayers only |
| Events | Public/family-open event discovery | No private events |
| Silent Prayer | Anonymous public prayer counter | Aggregate only |
| Join | Interest form | Consent required |
| Login | Entry to candidate/brother mode | No registration bypass |

## Mobile Candidate

| Section | Purpose | Key restrictions |
| --- | --- | --- |
| Dashboard | Candidate next steps | Candidate content only |
| Roadmap | Candidate path | No brother roadmap |
| Events | Candidate-visible events | No brother-only events |
| Announcements | Candidate communications | Candidate/public only |
| Contact | Assigned chorągiew/officer | Only assigned data |

## Mobile Brother

| Section | Purpose | Key restrictions |
| --- | --- | --- |
| Today | Daily summary | Personalized filtered data |
| Profile | Own membership identity | Critical data read-only |
| My Chorągiew | Local community context | No public brother list by default |
| Prayers | Public/brother/chorągiew prayers | Visibility enforced |
| Events | Relevant events | Participation intent only |
| Announcements | Relevant messages | No chat/comments |
| Formation | Roadmap and submissions | Officer decides approvals |
| Silent Prayer | Shared silent presence | Aggregate counters only |
| Settings | Notification preferences | No admin functions |

## Admin Lite

Admin Lite includes operational sections only: dashboard, chorągwie, brothers, candidate requests, candidates, prayers, events, announcements, roadmap requests/definitions, silent prayer events, and audit log.
