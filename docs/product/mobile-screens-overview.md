# Mobile Screens Overview

| Mode | Screen group | Required screens |
| --- | --- | --- |
| Public | Discovery | Public Home, About the Order, Login |
| Public | Prayers | Public Prayer Categories, Public Prayer List, Public Prayer Detail |
| Public | Events | Public Events List, Public Event Detail |
| Public | Join | Join Request Form, Join Request Confirmation |
| Public | Prayer presence | Public Silent Prayer |
| Candidate | Dashboard | Candidate Dashboard, Candidate Contact / Assigned Chorągiew |
| Candidate | Roadmap | Candidate Roadmap, Candidate Roadmap Step Detail |
| Candidate | Communications | Candidate Events, Candidate Announcements |
| Brother | Daily | Brother Today, Brother Profile, My Chorągiew |
| Brother | Content | Prayer Categories, Prayer List, Prayer Detail, Announcements List, Announcement Detail |
| Brother | Events | Events List, Event Detail |
| Brother | Formation | Formation Roadmap, Formation Step Detail, Roadmap Submission |
| Brother | Prayer presence | Silent Prayer List, Silent Prayer Session |
| Brother | Settings | Notification Preferences, Settings |

Implementation must use screen-state documentation in `/docs/screens/screen-state-model.md`.
