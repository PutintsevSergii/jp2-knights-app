# Acceptance Criteria

## Public Mode

- App opens without login.
- Guest can read public prayers.
- Guest can see public/family events.
- Guest can submit join request with consent.
- Guest can join public silent prayer anonymously.
- Guest cannot see private content.

## Candidate Mode

- Candidate can log in.
- Candidate sees dashboard, roadmap, events, announcements, assigned contact.
- Candidate cannot see brother-only content.

## Brother Mode

- Brother can log in.
- Brother sees Today, profile, chorągiew, prayers, events, announcements, formation, silent prayer.
- Brother cannot edit critical membership data.

## Admin Lite

- Officer manages own chorągiew data only.
- Super Admin manages all V1 data.
- Admin can manage candidate requests, candidates, brothers, events, prayers, announcements, roadmap submissions, silent prayer events.
- Critical changes are audited.

## Safety

- Private content is not public.
- Roadmap and candidate data are scoped.
- Silent prayer does not expose participant lists.
- Archive/deactivate is preferred over hard delete.
